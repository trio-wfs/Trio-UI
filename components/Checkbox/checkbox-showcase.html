Checkbox

Allows users to select one or more options from a set. Use for binary choices, multiple selections, or to show agreement to terms. Checkboxes are independent of each other within a group.

category Form Control verified Production Ready

rule Usage Guidelines

check_circle When to Use
Correct

  • Always include a clear, descriptive label
  • Use indeterminate for "Select All" parent checkboxes
  • Stack checkboxes vertically in a list
  • Use error color for validation failures
  • Binary opt-in choices (terms and conditions)

widgets Variants

Checkboxes support two color variants and an indeterminate state. Primary is the default. Error is used for validation failures or destructive confirmations.
