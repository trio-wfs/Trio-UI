ButtonGroup

Groups multiple related button actions into a single interface element.

Figma: 2172:9605 AHTG

Overview

ButtonGroup organizes multiple related actions into a compact, visually connected interface. Use for presenting actions in toolbars, modals, form footers, and page headers.

Properties

Property Type Options Default
variant VARIANT "contained", "outline" "contained"
size VARIANT "sm", "md" "sm"
color VARIANT "primary", "secondary" "secondary"
orientation VARIANT "horizontal", "vertical" "horizontal"
buttons Array 3-6 buttons (button-3 through button-8) 3 buttons

Size Variants

size="sm" // 32px height (8px × 4)

size="md" // 40px height (8px × 5)

AHTG Design System Compliance

Rule Implementation
Desktop Only ✅ No responsive patterns, fixed sizes
8px Spacing System ✅ Heights: 32px (sm), 40px (md) — multiples of 8
Primary Blue Usage ✅ Save/Update/Confirm actions only (#2196F3)
Secondary Usage ✅ Edit/Prompt/Manage actions (#F5F5F5)
Roboto Typography ✅ Inherited from MUI theme, 14px default
Material Icons ✅ Supported when used in individual buttons
Maximum Buttons ✅ 3-6 buttons (Figma spec: button-3 through button-8)

Color Semantics

Primary - Save/Confirm Actions

Use only for confirming or saving data. Examples: "Save & Continue", "Submit All", "Confirm Changes"

Secondary - Edit/Prompt Actions

Use for exploratory or editing actions. Examples: "Edit All", "Add Items", "Manage Options"
