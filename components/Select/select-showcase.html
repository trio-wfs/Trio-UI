Select

Allows users to choose a single option from a dropdown list. Use when you have 5 or more options that would clutter the interface as radio buttons.

category Form Control verified Production Ready

rule Usage Guidelines

check_circle When to Use
Correct
  • Use for 5+ options (otherwise use Radio buttons)
  • Sort options logically (alphabetically or by usage)
  • Provide a default selection when appropriate
  • Use clear, concise option labels
  • Include helper text for complex selections
cancel When NOT to Use
Incorrect
  • Don't use for 2-4 options (use Radio buttons instead)
  • Don't use for actions (use Button or Menu instead)
  • Don't use overly long option names
  • Don't change content based on selection without user action
  • Don't nest select dropdowns

straighten Sizes

Select supports two sizes. Medium (default) is used in standard forms. Small is used in dense layouts, toolbars, and inline filter controls where vertical space is limited.

Medium (default) — 38px input height
Small — 32px input height

widgets Variants

Select supports different configurations to accommodate various form contexts. The dropdown uses the custom Menu component for consistent styling across the application.

Default - With placeholder
Selected - With value chosen

toggle_on States

Select inputs have multiple interactive states to provide visual feedback. All state values are extracted directly from Figma specifications.

Default & Hover
Error - Validation failure
Disabled — gray fill + muted border (#E0E0E0)

recommend Best Practices

check_circle Do
Correct
  • Sort options in a logical order (severity, alphabetical, frequency)
  • Always include a visible label above the select
  • Use helper text to guide complex selections
  • Provide a meaningful placeholder (not just "Select")
  • Pre-select a default when one option is clearly most common
cancel Don't
Incorrect
  • Don't use Select for only 2-3 options — use Radio buttons instead
  • Don't use vague placeholders like "Select" or "Choose"
  • Don't hide critical information behind a dropdown
  • Don't use for triggering actions (use Button or Menu)
  • Don't nest select dropdowns inside each other
