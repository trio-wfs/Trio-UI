Toggle Button

A grouped strip of 3–8 icon buttons sharing a single primary-color border. Used in toolbars and breadcrumb bars for compact view or action switching.

infoOverview

ToggleButton groups 3–8 ButtonIcon slots into a visually unified strip. The shared blue border signals that the buttons belong to the same control group. Each slot is independently clickable; an active prop highlights the selected option with a light primary fill. Primary use: the view-toggle cluster at the right end of the Breadcrumb bar in the Page Header Toolbar.

ruleBest Practices

Do
  • Use 3 buttons as the default — filter, export, reset is the standard TRIO WFS pattern
  • Always mark one slot as active when used for view switching
  • Use sm in the breadcrumb bar and compact toolbars
  • Provide aria-label on every slot
Don't
  • Don't exceed 5 buttons — use a Menu or overflow pattern instead
  • Don't mix ToggleButton with standalone ButtonIcon in the same toolbar strip
  • Don't use for unrelated actions — slots should share a common theme
