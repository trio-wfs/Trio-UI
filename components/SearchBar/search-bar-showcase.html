Search Bar

A text input with an attached search icon button — used to filter lists, grids, or page content.

rule Usage Guidelines

check_circle When to Use
Correct
  • Use the Medium size by default
  • Reserve Small for dense toolbars (e.g., inline grid filters)
  • Provide a clear placeholder that tells the user what is searchable
cancel When NOT to Use
Incorrect
Using SearchBar for navigation
  • Don't use SearchBar for navigation or routing — it's a filter control
  • Don't use the Right type without reason — Left is the standard pattern
  • Don't use for global/site-wide search — use the App Bar search field instead

recommend Best Practices

check_circle Do
Correct
  • Use the Medium size by default. Reserve Small for dense toolbars (e.g., inline grid filters).
  • Provide a clear placeholder that tells the user what is searchable: "Search workers, shifts, or orders..." is better than just "Search".
cancel Don't
Incorrect
  • Don't use SearchBar for navigation or routing. It's a filter control, not a site-wide search.
  • Don't use the Right type without reason — Left is the standard pattern across TRIO WFS.
