Badge

Badges are small status indicators anchored to an element — used to show counts, notifications, or state without interrupting the user's flow.

category Atomic Component menu_book Storybook design_services Zeplin verified Production Ready

info Overview

psychology Purpose
Badges signal pending activity or counts without interrupting the user. In VMS workflows, they commonly appear on nav items, icons, and list items to indicate pending credentialing, timecard, or submission activity.
swap_horiz Types
Default — shows numeric counts (unread messages, pending items). 16px height pill shape.
Dot — simple presence indicator when count isn't meaningful. 8px diameter.
accessibility Accessibility
Badges use aria-label to announce counts to screen readers. Color alone does not convey meaning — pair with contextual text for assistive technology users.
