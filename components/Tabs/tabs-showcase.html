Tabs

Horizontal navigation that switches between related content panels on the same page.

info Overview

Tabs organize content into discrete sections. The active tab is visually indicated by a 2px primary-blue bar at the top of the tab, and the label shifts to medium weight blue. Inactive tabs use secondary gray text. Use tabs when users need to switch between a small number of related views without leaving the page.

view_week Variants

Four scroll variants control whether arrow buttons appear when tabs overflow the container.

Tab Group Default
No scroll arrows. All tabs visible. Use when tab count fits the container.
Left Scroll
Left arrow only — for end-of-list navigation.
Right Scroll
Right arrow only — for beginning-of-list navigation.
Left and Right Scroll
Both arrows — use when many tabs may overflow the container.

toggle_on States

Each tab can be in one of three states.

Active
Primary blue label, medium weight, 2px blue indicator at top.
Inactive
Secondary gray label (#757575), regular weight, no indicator.
Hover
Background shifts to action-hover (rgba 0,0,0,0.04), label darkens slightly.
Disabled
Disabled gray label (#9E9E9E), not interactive, cursor not-allowed.

add_photo_alternate With Icons

Individual tabs support an optional left icon, right icon, or both. Icons scale with the tab height — 18px by default.

thumb_up Best Practices

✓ Do

Use tabs for parallel content sections where only one is visible at a time — worker profile sections, order detail views.

✗ Don't

Don't use tabs for sequential workflows (use a stepper) or for content that must all be seen together.

✓ Do

Keep tab labels short (1–2 words). Use title case. The active state makes the selection clear without needing long labels.

✗ Don't

Don't exceed 8 tabs. Beyond 8, consider a secondary navigation or grouping. Cognitive load increases significantly.

✓ Do

Use scroll variants when the number of tabs is variable or context-dependent (e.g., user-configured views).

✗ Don't

Don't put tabs inside modals unless absolutely necessary. Modals have limited height — tabs can make content harder to scan.
