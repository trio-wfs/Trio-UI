Checkbox

Allows users to select one or more options from a set. Use for binary choices, multiple selections, or to show agreement to terms.

Overview

info Purpose
Checkboxes let users select one or more items from a list, or turn an option on or off. Use when:
  • Users can select multiple options
  • Selections are independent of each other
  • Users need to opt in or agree to terms
tune States
Checked: true, false
Indeterminate: true (partial), false
Disabled: true, false
Color: primary, error
Hover: on, off
accessibility Accessibility
  • Keyboard accessible (Space to toggle)
  • Always include a visible label
  • Group related checkboxes with fieldset/legend
  • Use aria-label if no visible label

rule Usage Guidelines

check_circle When to Use
  • Use for multiple independent selections
  • Always include a clear label
  • Use indeterminate for "select all" parent checkboxes
  • Stack checkboxes vertically in a list
  • Use error color for validation failures
cancel When NOT to Use
  • Don't use for mutually exclusive options (use Radio instead)
  • Don't use more than one checkbox for a single choice
  • Don't use for on/off toggles (use Switch instead)
  • Don't nest checkboxes more than one level deep
  • Don't change content based on checkbox state without user action
