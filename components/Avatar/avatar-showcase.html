Avatar

Identity primitive for user and agency authors. The anonymized variant powers cross-agency privacy in shared threads.

account_circle Overview

Circular identity primitive rendering 1–2 initials over a colored fill. Non-interactive — no hover, focus, or disabled states by design. Two sizes (24px / 32px) and eight color variants cover identifier rendering across user and agency posts.

palette Color Variants

Eight colors, each mapped to an existing TRIO design token. Choose based on author role: staff is the slate anchor for TRIO Staff/RMs, anonymized is reserved for cross-agency privacy, others are identifier choices (typically name-hashed in code).

lightbulb Best Practices

  • Staff use color="staff" always. Slate anchors Staff identity across all threads so users find them instantly.
  • Hash identifiable agency colors from a stable name input. Same author → same color across sessions. Excludes staff (reserved) and anonymized (reserved).
  • Use color="anonymized" for cross-agency rendering. Three anonymized agencies looking identical is a feature — it visualizes the privacy guarantee.
  • Don't add hover or click states. Avatar is a display primitive. Interactive author surfaces (e.g. profile popovers) wrap the Avatar in a separate interactive component.
  • 1–2 character initials only. Longer strings will overflow the circle; truncate before passing.
