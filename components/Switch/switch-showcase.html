Switch

Toggle between two states (on/off). Use for immediate actions that take effect instantly without requiring a save or submit action.

category Form Control

check_circle Production Ready

Overview

info Purpose
Switches toggle between on/off states with immediate effect. Use when:
  • Changes take effect immediately (no save/submit needed)
  • Toggling a single setting or feature
  • The change is reversible
tune States
State: on, off
Disabled: yes, no
Label placement: left, right, top
Size: 28x16px track, 12px thumb
accessibility Accessibility
  • Keyboard accessible (Space to toggle)
  • Always include a visible label
  • Use clear on/off states visually
  • Consider using aria-label if no visible label

rule Usage Guidelines

check_circle When to Use
  • Use for immediate on/off toggles
  • Include clear labels describing what's being toggled
  • Use for settings that take effect instantly
  • Make the current state obvious
  • Use for reversible actions
cancel When NOT to Use
  • Don't use if changes require save/submit (use Checkbox instead)
  • Don't use for mutually exclusive options (use Radio instead)
  • Don't use for actions (use Button instead)
  • Don't use without a label
  • Don't use for irreversible actions
