Navigation Header

The top-level application header for all TRIO WFS pages. Contains the brand bar with search and user info, plus the horizontal navigation bar below.

info Overview

The NavigationHeader is a two-row component that sits at the very top of every TRIO WFS page. Row 1 is the brand bar (blue-grey) with the TRIO logo, environment badge, search, and user avatar. Row 2 is the white navigation bar with horizontal menu items.

lightbulb Best Practices

  • The NavigationHeader is always the topmost element on every TRIO page — nothing sits above it.
  • Nav items with dropdowns show keyboard_arrow_down. Items without dropdowns have no arrow.
  • Only one nav item can be active at a time. The active indicator is a 2px bottom border.
  • The environment badge (TRIO WIP, TRIO PROD) should be hidden in production or set to a neutral label.
  • The SearchBar in the brand bar uses the existing SearchBar component with type="Right".
  • The brand bar background (#607D8B) is the only place this color appears in the application — do not reuse it elsewhere.
