🛰️ Shift Distribution Modal

Interactive showcase of the shift distribution component with privileged provider counts. Select audiences, create schedules, and toggle notifications per row.

Features

Provider Counts

Display privileged provider counts per audience type. Resource pools show simple counts, tiers show context (7 with providers / 10 total).

"Also Notify Now" Toggle

Per-row independent toggles. Default ON. Disabled when provider count = 0 to prevent sending notifications to empty audiences.

AHTG Compliant

Built with AHTG design tokens only. No invented colors, spacing, or components. Uses Inter typography and MUI.

💡 Key Implementation Details

Provider Counts: Resource pools show "(N providers)". Tiers/Agencies show "(N with Providers / M Total)" for context clarity.

Toggle States: Defaults ON. Disabled when provider count = 0 to prevent sending notifications to unavailable audiences.

Per-Row Control: Each schedule row has independent toggle state management. Changes don't affect other rows.
