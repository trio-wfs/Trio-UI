Menu

Displays a list of choices on a temporary surface. Use when you need to provide users with multiple options or actions in a compact, contextual way.

info Overview

psychology Purpose
Menus display a list of choices on temporary surfaces. Use when showing contextual actions, providing navigation in a compact space, or organizing related actions together.
tune Variants
Single — simple action list.
Grouped — sections with titles and dividers.
Multi — items with selection states.
Min width: 254px. Item height: 37px.
accessibility Accessibility
Keyboard navigation (Arrow keys, Enter, Escape). Focus management returns to trigger on close. ARIA roles and labels. Screen reader announcements for state changes.
