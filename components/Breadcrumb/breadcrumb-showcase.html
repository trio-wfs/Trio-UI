Breadcrumb

Navigation path strip showing the user's location within a hierarchy. Sits between the Page Header Toolbar and the main content area. Supports an optional record count and action buttons on the right.

infoOverview

The Breadcrumb is a 40px-tall strip with a light gray background. It renders an ordered list of navigation links with the current page shown in dark text with a blue bottom underline. Two states differ in divider character. The right side optionally shows a record count and a ToggleButton strip (filter/export/reset).

ruleBest Practices

Do
  • Always mark exactly one item as selected — the current page
  • Keep link labels short — use the page/section name only
  • Use the / state as default; | for tab-style navigation
  • Show record count when the page is a list or table view
Don't
  • Don't exceed 6–7 path segments — truncate middle items with ellipsis
  • Don't put non-navigation actions in the breadcrumb strip (filter belongs in ToggleButton, not a standalone button)
  • Don't use breadcrumb for step-by-step wizard navigation — use a Stepper
